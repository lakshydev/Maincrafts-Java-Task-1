# Task 1: Full-Stack Landing Page with Spring Boot

This project is a full-stack web application developed as part of the **Maincrafts Technology Internship**. It features a modern, responsive landing page that collects user data and sends it to a Java backend.

## Features
- **Modern UI:** A clean and professional landing page built with HTML5 and CSS3.
- **Form Integration:** A functional contact form (Name, Email, Message).
- **Backend Processing:** A Spring Boot REST Controller that handles POST requests.
- **Console Logging:** Real-time printing of submitted data to the server terminal.

## Tech Stack
- **Frontend:** HTML5, CSS
- **Backend:** Java, Spring Boot (Web)
- **Build Tool:** Maven
